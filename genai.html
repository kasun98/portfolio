Gen AI

An Advanced Approach to Research and Data Interaction


I've developed a cutting-edge generative AI RAG web app, utilizing some of the most advanced LLM models and technologies available today. This platform offers various capabilities for interacting with PDFs, CSV files, and providing general information.

GitHub GitHub Repository

Key Features

The web app leverages state-of-the-art models and tools to deliver exceptional performance and user experience.

LLM Models Used: Ollama-Gemma 2b, Llama3 (8b & 70b), Gemini-pro, Gemma 7b, Mixtral

Technologies & Tools: Pandas AI, LangChain, LangSmith, Groq Inference Engine, Streamlit

Bots and Their Capabilities

PDF Document Chat Bot

This bot allows users to chat with multiple PDF documents. By leveraging the Groq LPU Inference Engine, it delivers fast and accurate responses.

CSV Data Analysis Bot

Designed to analyze CSV data files, this bot uses Pandas AI and Smart DataFrame technologies, coupled with LLM models, to provide insightful data analysis and visualizations.

General Information Chat Bot

This bot is built using the Ollama-Gemma:2b model, which has been locally downloaded and fine-tuned to deliver accurate and reliable information.

Development Stack

  • Ollama-Gemma:2b, Llama3 (8b & 70b), Gemini-pro, Gemma 7b, Mixtral LLM models
  • Pandas AI for data processing and analysis
  • Groq Inference Engine for fast response times
  • LangChain, LangSmith LLMOps
  • Streamlit for creating a seamless and interactive user experience
  • Huggingface: For deployment of the end to end web application.

Performance and Results

The web app has demonstrated exceptional performance in various tasks. The PDF and CSV bots provide quick and accurate responses, enhancing research and data analysis processes significantly.

Limitations and Challenges

Despite its impressive performance, the web app does have some limitations:

  • API Restrictions: The free API services used may occasionally hit request limits, causing temporary disruptions.

Conclusion

This generative AI RAG web app represents a significant advancement in research and data interaction capabilities. With powerful LLM models and innovative technologies, it offers users enhanced research, data analysis, and general information retrieval functionalities.
